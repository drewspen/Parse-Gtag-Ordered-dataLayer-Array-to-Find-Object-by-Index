# Parse-Gtag-Ordered-dataLayer-Array-to-Find-Object-by-Index
Parse Gtag Ordered dataLayer Array to Find Object by Index

This Google Tag Manager variable template uses Sandbox JavaScript to find the most recent matching dataLayer events formatted as "0: event" and "1: purchase" (for example). It parses the arguments object / internal array structure used by the gtag() function. This is also referred to as command queue syntax. This differs from the dataLayer event structure "event: purchase". The gtag() function is a wrapper that pushes information into the dataLayer. When you call gtag('event', 'purchase', {...}), the function takes those individual arguments and pushes them into the dataLayer array as a set of indexed values rather than a single flat object. The variable template returns the JSON object "payload" identified by chosen index. To avoid unnecessary errors in the browser console, the GTM {{Event}} variable is passed also so that the dataLayer is parsed only when {{Event}} equals the identified value of index 1.
